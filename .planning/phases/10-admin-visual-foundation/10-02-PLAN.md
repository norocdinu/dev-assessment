## Design Token Reference Map

Use this map for every substitution in all tasks below. Apply it mechanically — do not paraphrase or approximate.

### Gray → Neutral token substitutions:
| Old class | New class |
|-----------|-----------|
| `bg-white` | `bg-card` |
| `bg-gray-50` (page background) | `bg-background` |
| `bg-gray-50` (table header / subtle area) | `bg-muted/10` |
| `bg-gray-50/50` (alternating table row) | `bg-muted/5` |
| `bg-gray-100` | `bg-muted/20` |
| `hover:bg-gray-50` | `hover:bg-muted/10` |
| `hover:bg-gray-100` | `hover:bg-muted/20` |
| `border-gray-200` | `border-border` |
| `border-gray-100` | `border-border/50` |
| `border-gray-300` | `border-border` |
| `text-gray-900` | `text-foreground` |
| `text-gray-700` | `text-foreground/80` |
| `text-gray-600` | `text-foreground/70` |
| `text-gray-500` | `text-muted` |
| `text-gray-400` | `text-muted/70` |
| `divide-gray-100` | `divide-border/50` |
| `bg-gray-50 cursor-not-allowed` | `bg-muted/10 cursor-not-allowed` |

### Blue → Brand token substitutions:
| Old class | New class |
|-----------|-----------|
| `bg-blue-50 text-blue-700` | `bg-[rgb(var(--brand-rgb))]/10 text-[var(--brand)]` |
| `text-blue-600` | `text-[var(--brand)]` |
| `text-blue-700` | `text-[var(--brand)]` |
| `bg-blue-600` | `bg-[var(--brand)]` |
| `hover:bg-blue-700` | `hover:bg-[var(--brand)]/90` |
| `border-t-blue-600` | `border-t-[var(--brand)]` |
| `hover:bg-blue-50` | `hover:bg-[rgb(var(--brand-rgb))]/10` |
| `bg-blue-500` | `bg-[var(--brand)]/80` |
| `bg-blue-100 text-blue-700` | `bg-[rgb(var(--brand-rgb))]/10 text-[var(--brand)]` |
| `bg-blue-50 text-blue-600` | `bg-[rgb(var(--brand-rgb))]/10 text-[var(--brand)]` |

### PRESERVED — do NOT migrate these (semantic status colors):
- `bg-green-*`, `text-green-*`
- `bg-red-*`, `text-red-*`
- `text-orange-*`
- `bg-yellow-*`, `border-yellow-*`, `text-yellow-*`
- `bg-amber-*`, `text-amber-*`
