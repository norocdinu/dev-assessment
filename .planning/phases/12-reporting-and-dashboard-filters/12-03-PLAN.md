      <action>
Open apps/web/src/app/globals.css.

Append the following block at the end of the file (after the closing `}` of `@layer base`):

```css
@media print {
  aside { display: none !important; }
  .md\:hidden.sticky { display: none !important; }
}
```

The backslash in `.md\:hidden` is the CSS escape for the colon in the Tailwind class name `md:hidden`. This hides the desktop sidebar (rendered as `<aside>`) and the mobile sticky top-bar (which carries both `md:hidden` and `sticky` classes).

Do not remove or modify any existing CSS. The appended block must be at the very end of the file.
