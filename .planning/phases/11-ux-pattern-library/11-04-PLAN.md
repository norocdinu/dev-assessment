## Task 2: Submission detail page — replace window.confirm and alert() with ConfirmDialog and toast

<action>
Make the following targeted edits to `apps/web/src/app/(admin)/submissions/[linkId]/page.tsx`:

**Important structural note:** The current file defines `handleDelete` at line ~133, AFTER the `if (loading)` and `if (error)` early-return blocks (lines ~117–128). The new `requestDelete` and `executeDelete` functions must be placed BEFORE those early returns (alongside the `useEffect`) to ensure they are in scope when the main JSX renders. `useState` hooks must always be before any early returns.

### 1. Add imports
Add after the existing imports:
```tsx
import { ConfirmDialog } from '@/components/ui/ConfirmDialog';
import { toast } from 'sonner';
```

### 2. Add state for confirm dialog — BEFORE early returns
Add after the existing state declarations (after `const [deleting, setDeleting] = useState(false);` — this is still BEFORE the `if (loading)` early return, so placement is correct):
```tsx
const [confirmDelete, setConfirmDelete] = useState(false);
```

### 3. Move handler functions before early returns

The file currently has `handleDelete` defined AFTER the early-return blocks (line ~133). Move the replacement functions to BEFORE the `if (loading)` block. Place them immediately after the `useEffect` and state declarations:

```tsx
  function requestDelete() {
    setConfirmDelete(true);
  }

  async function executeDelete() {
    setConfirmDelete(false);
    setDeleting(true);
    try {
      await api.delete(`/admin/submissions/${linkId}`);
      router.push('/submissions');
    } catch {
      setDeleting(false);
      toast.error('Failed to delete submission. Please try again.');
    }
  }
```

Remove the old `handleDelete` function entirely (it is defined after the early returns and uses `window.confirm` + `alert`).

### 4. Update the Delete button call site
Find the button that called `onClick={handleDelete}` (around line 177) and change it to `onClick={requestDelete}`.

### 5. Add ConfirmDialog to the main JSX return

Add the ConfirmDialog just before the final closing element of the main return block (the last `</div>`):
```tsx
      <ConfirmDialog
        open={confirmDelete}
        title="Delete submission?"
        description="This will permanently remove this candidate's results and cannot be undone."
        confirmLabel="Delete"
        destructive
        onConfirm={executeDelete}
        onCancel={() => setConfirmDelete(false)}
      />
```
