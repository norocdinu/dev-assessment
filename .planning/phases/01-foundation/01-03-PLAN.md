## Objective

Build the admin frontend: login page, authenticated route guard, question bank CMS (list with search/filter, create, edit, archive, version history), and test configuration management UI. Uses Next.js 14 App Router with a `(admin)` route group.

## Context

- Next.js 14 App Router, TypeScript
- All admin pages live under `app/(admin)/` route group
- Auth state: stored in a `user` cookie (set by API) read server-side in `layout.tsx`; redirects to `/login` if missing
- API calls: centralised Axios instance at `src/lib/api.ts` pointing to `NEXT_PUBLIC_API_URL`
- Styling: Tailwind CSS (add to project in this plan)
- Tables: TanStack React Table v8
